[h1]What's Included:[/h1]
[h3]Speakers:[/h3]
[list]
    [*]"JBL" Style Large Speaker
    [*]Long/Tall Speaker
    [*]Loud Speaker
    [*]"Bose" Style Planter Speaker
    [*]Small Speaker
[/list]
[h3]Features:[/h3]
[list]
    [*]Multiple textures able to be recolored.
    [*]Speakers can be rotated upon their mounts.
    [*]Choose any MP3 or OGG file to play on loop from the speaker.
    [*]Choose any sound to play on loop during a specified coaster's E-Stop.
    [*][NEW!] Choose any sound to play on-queue for a specified coaster trigger.
    [*]All basic audio settings, like range, volume, pitch, etc, can be adjusted via object properties.
[/list]

The models are meant to be simple, relatively low-poly, and do not have massively detailed textures in effort to keep them efficient. Comment below if there is a specific speaker style you would like to see added to the pack!

[hr][/hr]
[h1]How to Extract:[/h1]
[list]
    [*]Open the park in your editor.
    [*]Click on File > Extract Package, and choose where you want to save the park.
    [*]Open the park in your File Explorer. Then open the scenery folder.
    [*]Copy the reds_speakers folder to any park of your own that you want to use it with.
[/list]
